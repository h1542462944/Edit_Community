Edit_Community
==========
# 软件介绍
这是一款**作业展示器**软件,能够实现在班级教室电脑展示当天的功能.
# 颜色声明
+ 第一层 属性
	+ <font color=red>important</font> 比较重要的信息.
	+ <font color=seagreen>improve</font> 对功能的优化.
	+ <font color=yellow>debug</font> 对bug的修复.
	+ <font color=orange>bug</font> 已发现但是未解决的bug.
	+ <font color=gray>delete</font> 已删除的功能.
+ 第二层 内容
	+ <font color=orange>main</font> 主要功能.
	+ <font color=seagreen>brush</font> 笔刷图层.
	+ <font color=gray>settings</font> 设置的功能.
	+ <font color=yellow>extension</font> 相关功能的扩展.
	+ <font color=skyblue>window</font> 对窗体功能的优化.
